進位轉換器 (0-255)

這是一個使用 Python Tkinter 開發的圖形化介面 (GUI) 工具，專門用於進行 10 進位、2 進位與 16 進位之間的互相轉換。
使用 Tkinter 的理由（介面開發）：
本程式選用 tkinter 作為 GUI 框架，主因其為 Python 的標準內建庫，具有跨平台相容性高且無需額外安裝套件的優點，適合用於展示具備互動功能的工具，並不影響核心轉換邏輯的獨立性。

1.實作重點 (符合規範)

手寫轉換邏輯：本程式完全沒有使用 Python 內建的 bin() 或 hex() 函式。

演算法核心：

10 轉 2：透過「除以 2 取餘數」的循環邏輯實現，並自動補足至 8 位元。

10 轉 16：使用自定義對照表與「除以 16 取餘數」邏輯實現。

異常處理：設有 try-except 機制，能攔截非數字輸入，並限制輸入範圍在 0 至 255 之間。

2.執行環境

語言: Python 3.x

套件: Tkinter (內建)
